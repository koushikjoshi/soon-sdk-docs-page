# Transactions

### `sendTransaction(senderPrivateKey, receiverPublicKey, amount, client)`
Sends a SOL transaction.

#### Parameters:
- **senderPrivateKey**: The sender's private key.
- **receiverPublicKey**: The receiver's public key.
- **amount**: Amount in SOL.
- **client**: An instance of `SOONClient`.

### Example
```javascript
const txSignature = await Transactions.sendTransaction(
  'SenderPrivateKey',
  'ReceiverPublicKey',
  0.01,
  client
);
console.log('Transaction Signature:', txSignature);
